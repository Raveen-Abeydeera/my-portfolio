Project Overview

Greenlife Wellness is a web-based platform designed to manage the daily operations of a wellness center. It allows clients to browse services, view therapist profiles, and book appointments online. The system includes a secure client dashboard for managing bookings and profile settings.

Key Features

  • User Authentication: Secure login and registration for clients.
  • Appointment Booking: Real-time booking system for therapy sessions.
  • Therapist Management: Profiles and availability for staff.
  • Admin Dashboard: Backend interface for managing services and bookings.
  • Responsive Design: Optimized for both desktop and mobile devices.

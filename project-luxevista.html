Project Overview

LuxeVista Resort is a native Android application designed to enhance the guest experience by providing a seamless platform for managing their stay. It features a robust room booking system with date-based availability filtering, service reservations (Spa, Dining), and a secure user account system.

Key Features

  • Native Performance: Built with Java and Android SDK for a smooth, premium feel.
  • Smart Booking Engine: Filter rooms by date availability using complex SQL queries.
  • Service Reservations: Book spa treatments and dining experiences directly in-app.
  • Local Persistence: Uses SQLite for robust offline data storage and retrieval.
  • Secure Auth: User registration and login system with input validation.
